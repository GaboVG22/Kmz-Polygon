# Desplegar en Streamlit Cloud

GitHub solo almacena el código. Para que se vea como una aplicación web, use Streamlit Cloud.

## Pasos

1. Suba la carpeta completa a GitHub.
2. Entre a https://streamlit.io/cloud
3. Cree una nueva app.
4. Seleccione:
   - Repository: su repositorio
   - Branch: main
   - Main file path: app.py
5. Presione Deploy.

## Error frecuente

Si ve una página con el texto del README, está viendo el repositorio o una página estática, no la aplicación.
